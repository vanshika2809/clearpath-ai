# ClearPath AI 

ClearPath AI is a beginner-friendly hackathon prototype for indoor accessibility navigation.

## Features
- Step-free routing demo (avoids stairs by default)
- Voice guidance (Text-to-Speech)
- Speech input (Speech-to-Text, if browser supports it)
- Community accessibility reports (saved to localStorage)
- Clean high-contrast UI

## Run locally
1. Install Node.js (v18+ recommended)
2. Install dependencies:
   ```bash
   npm install
3.Start dev server:
   ```bash
npm run dev
